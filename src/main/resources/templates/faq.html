Frequently Asked Questions

  1. What is Cabalchan?

    Cabalchan is an anonymous internet forum where users can post text, images, videos, files, and more.

  2. How does post formatting work?

    >text makes it a orangetext quote: >text.
    ==text== makes it a redtext heading: text
    **text** makes it spoiler: text

  3. What attachment file types are supported?
    • Images - png, jpg/jpeg, gif
    • Video - webm, mp4
    • Files - pdf, zip
    • Audio - mp3, ogg(vorbis), wav
  4. What is the file upload limit?

    Files may not be larger than 15MB.
